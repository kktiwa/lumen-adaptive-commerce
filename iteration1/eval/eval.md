# Evaluation Criteria

LLM-as-judge scoring for personalized product description sections. Each section is scored 1–5 per criterion; **overall** is the average of the 4 criterion scores.

## Section Criteria

| Section | Description | Criterion 1 | Criterion 2 | Criterion 3 | Criterion 4 | Scale |
|---------|-------------|-------------|-------------|------------|-------------|-------|
| **Header** | Main headline of the product description | Persona alignment | Value proposition | Specificity | Memorability | 1–5 |
| **Subheader** | Secondary headline supporting the header | Complementarity | Secondary benefit | Natural flow | Conciseness | 1–5 |
| **Tagline** | Short, memorable brand statement | Brevity | Emotional resonance | Brand voice | Memorability | 1–5 |
| **Summary** | Main product description paragraph | Faithfulness | Feature coverage | Persona relevance | Readability | 1–5 |
| **Visual Cues** | Design and aesthetic descriptors | Specificity | Actionability | Brand coherence | Originality | 1–5 |
| **CTA** | Call-to-action button text | Action verb strength | Urgency / personalization | Funnel alignment | Clarity | 1–5 |
| **Metatags** | SEO meta title and description | Length compliance | Keyword relevance | No keyword stuffing | Click-worthiness | 1–5 |
| **Social Proof** | Reviews, ratings, or testimonial copy | Specificity | Authenticity | Relevance | Tone consistency | 1–5 |
| **Hyperlink Label** | Anchor text for embedded links | Descriptiveness | Accessibility | Contextual fit | Conciseness | 1–5 |

## Criterion Definitions

| Criterion | Definition |
|-----------|------------|
| Persona alignment | Speaks directly to a specific user segment or persona |
| Value proposition | Core product benefit is immediately clear |
| Specificity | Avoids vague/generic language — feels tailored not templated |
| Memorability | Would stop someone while scrolling; catchy and distinctive |
| Complementarity | Expands on the header without repeating it |
| Secondary benefit | Introduces a new angle or benefit |
| Natural flow | Reads naturally after the header |
| Conciseness | Appropriately brief and scannable |
| Brevity | Under 10 words and punchy |
| Emotional resonance | Evokes a feeling or aspiration |
| Brand voice | Tone feels consistent and intentional |
| Faithfulness | All claims grounded in plausible product facts (no hallucinated specs) |
| Feature coverage | Covers key product benefits comprehensively |
| Persona relevance | Language and focus tuned to the target user |
| Readability | Well-structured, engaging, free of filler |
| Actionability | Could a designer directly implement these cues? |
| Brand coherence | Cues align with the product's tone and persona |
| Originality | Feels distinctive rather than templated |
| Action verb strength | Starts with a strong, clear verb |
| Urgency / personalization | Creates motivation or feels personalized |
| Funnel alignment | Matches the appropriate stage (awareness vs. purchase) |
| Clarity | Next step unmistakable to the user |
| Length compliance | Title ≤60 chars, Description ≤160 chars |
| Keyword relevance | Primary keywords naturally included |
| No keyword stuffing | Reads naturally, not robotically |
| Click-worthiness | Would appear compelling in a search result |
| Authenticity | Feels credible and verifiable — not manufactured |
| Tone consistency | Matches the voice of the surrounding copy |
| Descriptiveness | Label describes the destination clearly |
| Accessibility | Avoids generic labels like "click here" or "learn more" |
| Contextual fit | Integrates naturally into surrounding copy |
